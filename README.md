# HR Analytics Project

## Project Objective
To analyze employee data and identify key factors influencing employee attrition,
salary trends, department performance, and workforce diversity using SQL, Python, and Power BI.

## Tools Used
- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI
- GitHub

## Key Analysis Areas
- Employee attrition analysis
- Department-wise performance
- Salary vs experience
- Gender diversity metrics
